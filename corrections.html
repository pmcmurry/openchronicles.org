Corrections

Every Open Chronicles edition is versioned. Errors are corrected in new versions and logged here.

Report an error

If you find a mistake in any published edition — a mistranslation, a wrong name, date, or number, a typographical fault, a broken reference — please report it on the project's issue tracker (a free GitHub account is required). An email contact will be added here shortly.

A useful report names:

  • the work and edition (source-faithful or reader’s);
  • the version (printed in the colophon and page footer);
  • the location — folio marker for source-faithful editions, chapter for reader’s editions, or a PDF page number;
  • what you believe is wrong, and — if you can — why.

Substantive corrections are verified against the source witnesses before they are applied. Corrected editions receive a new version number; the change is recorded below and in the edition's colophon history.
