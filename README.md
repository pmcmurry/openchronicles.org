# openchronicles.org

Static public site for **Open Chronicles**.

## Local preview

```bash
python -m http.server 8080
```

Open http://localhost:8080

## Deploy

GitHub Pages from this folder (`main` branch root or `/docs`—see repo settings).

Custom domain: `openchronicles.org` (see `CNAME`).

## Pages

| File | Role |
|------|------|
| `index.html` | Home (no works yet) |
| `library.html` | Empty library |
| `method.html` | Editorial method |
| `license.html` | CC BY 4.0 |
| `about.html` | About |
| `branding.html` | Wordmark / palette / logo options |
