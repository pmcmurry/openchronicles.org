HOW TO PREVIEW THE SITE LOCALLY
===============================

1) BEST — local server
   In PowerShell:

   cd C:\historiens_occidentaux\project\website
   python -m http.server 8080

   Then open:
   http://127.0.0.1:8080/

   Branding archive (internal):
   http://127.0.0.1:8080/branding.html

2) ALSO WORKS — open files directly
   In File Explorer go to this folder and double-click index.html.
   Relative links work; some browser security rules may block fonts when
   using file:// URLs.

3) PUBLIC SITE
   https://openchronicles.org

   See DNS.md for GitHub Pages + Cloudflare setup.
